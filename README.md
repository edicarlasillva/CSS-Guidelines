## Terminologia

### Regra

Regra é o nome dado ao seletor (ou grupo de seletores) acompanhados de um grupo de propriedades com seus respectivos valores.

Segue um exemplo:

```css
.titulo {
  font-size: 18px;
  line-height: 1.2;
}
```
